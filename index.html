About my Fractals Forest

    Since we have many beautiful forests in Costa Rica, I decided to create my own, code-genereated forest. Everything you see on the above Canvas are dots and lines, generated with javascript code. No images were used to create on my digital forest, I'm planning on keeping it like that.

    Since 2013 I began researching about fractals and how to implement fractal trees. My main reasons for creating this experiments using Javascript are that I use it on my regular (Front End Developer) job and it also makes it easier to share things directly on the browser. Whenever possible, I try to use plain javascript.

    After repeating code lots of times and reusing my own code snippets, I wrote my own javascript library for the things I need. So far (January 2015) it draws the basic things on the screen for me and renders the Canvas. You can take a look at http://juulio.com/js/libs/canvasElements.js

    Since 2014, I've created many individual fractal trees, some of them with leaves and some even have fruits. Most of their values are randomized so one tree will never look exactly like the others. They have different parameters for their angles, depth, height, colors and the probability of drawing a leaf or a fruit. You can check some of those experiments at the experiments section.

    There are 3 main components on this Canvas: The background, the trees and the clinging plant. All of them were created using only javascript, looking for examples on the web, learning the HTML5 Canvas API (super cool and interesting by the way)

    The background has two main areas: the sky and the grass. It was generated using Canvas gradients on a function called "drawSkyAndGrass".

    {% highlight javascript %} var drawSkyAndGrass = function() { context.save(); context.globalAlpha = 0.2; // First create a CanvasGradient object. var linGrad = context.createLinearGradient(0, 0, 0, canvas.height); linGrad.addColorStop(0, '#00BFFF'); Light Blue linGrad.addColorStop(0.45, 'white'); // White in the middle linGrad.addColorStop(0.85, '#55dd00'); // Green context.fillStyle = linGrad; // The canvas and the gradient have the same area context.fillRect(0, 0, canvas.width, canvas.height); context.restore(); }; {% endhighlight %}

    We have many beaches in Costa Rica and a warm weather almost all the year round. On December 2015 I visited Playa Dominical on the Pacific Coast. There were many large trees with giant clinging plants. That's where I got the idea of adding a clinging plant to my fractal trees. I was riding my car and saw it and thought "This thing can be drawn using a recursive function".
